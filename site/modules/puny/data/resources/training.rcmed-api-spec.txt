# RCMED API

The RCMED API is a REST-based web service allow for space/time query and retrieval of observational and model data from the cloud-based data storage. 

We keep an updated version of the API, dynamically linked to the database parameters [at this URL](http://rcmes.jpl.nasa.gov/query-api/query.php).

A static version of the API is provided below, for a quick look at its major parameters and return types.

## Overview
This service provides a REST-based access to the Regional Climate Model Evaluation database. To request data, please formulate a query as follows:

`http://rcmes.jpl.nasa.gov/query-api/query.php?datasetId=#Â¶meterId=#&latMin=#&latMax=#&lonMin=#&lonMax=#&timeStart=YYYYMMDDTHHMMZ&timeEnd=YYYYMMDDTHHMMZ`

## Query Parameters

    datasetId
        The unique id of the requested dataset. See Datasets for complete list
    parameterId
        The parameter (SurfAirTemp_A, etc) to return values for. See Parameters for complete list
    latMin
        The latitude of the lower-left corner of the results bounding box (minimum desired latitude)
    latMax
        The latitude of the upper-right corner of the results bounding box (maximum desired latitude)
    lonMin
        The longitude of the lower-left corner of the results bounding box (minimum desired longitude)
    lonMax
        The longitude of the upper-right corner of the results bounding box (maximum desired longitude)
    timeStart
        The ISO time marking the beginning of the desired time range (Format: YYYYMMDDTHHMMZ)
    timeEnd
        The ISO time marking the end of the time range (Format: YYYYMMDDTHHMMZ)
